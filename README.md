# VueJS-Projects

# ```1. Add DC Heros```

Clone This Project (Make Sure You Have Git Installed)
```
https://github.com/Alchemist98/VueJS-Projects.git
```

Install VueJS in your system
```
npm install -g @vue/cli
```

```
cd VueJS-Projects
cd DC Heros
```

Run the project
```
npm run serve
```

Your project will run on localhost
```
http://localhost:8080/
```
